# X Connect Importados PY

Uma plataforma de catálogo digital desenvolvida para facilitar a apresentação de produtos importados e simplificar o processo de atendimento ao cliente.

A proposta combina uma interface moderna com navegação intuitiva, busca rápida de produtos, filtros inteligentes e integração direta com WhatsApp, permitindo que o usuário encontre o produto desejado e entre em contato com a loja em poucos cliques.

## ✨ Recursos

* Catálogo completo de produtos
* Busca e filtros avançados
* Cotação do dólar integrada
* Página detalhada de produtos
* Integração com WhatsApp
* Design responsivo
* Experiência inspirada nos maiores marketplaces de importados da América do Sul

## 🚧 Status do Projeto

Protótipo funcional desenvolvido para validação de layout, navegação e experiência do usuário junto ao cliente.

## 🎯 Objetivo

Criar uma experiência de compra simples e eficiente para clientes que desejam consultar produtos, verificar preços e iniciar o atendimento diretamente pelo WhatsApp da loja.

## 📱 Fluxo de Uso

1. O cliente acessa o catálogo.
2. Pesquisa ou filtra produtos por categoria.
3. Visualiza os detalhes do produto.
4. Consulta a cotação do dólar.
5. Envia uma solicitação diretamente para o WhatsApp da loja.
